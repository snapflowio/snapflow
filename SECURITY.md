# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Snapflow, please report it responsibly by emailing **security@snapflow.io**.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We will acknowledge this within 48 hours and work with you to understand and address the issue.

## Note

We do not currently have a bug bounty program. We're a small team building tools for the community, and we appreciate your responsible disclosure to help keep Snapflow secure for everyone.
